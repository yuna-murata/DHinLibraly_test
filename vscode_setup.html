VSCodeでTEIを編集する環境を整える

前提

  • XMLファイルの編集には専用のエディタ（編集ソフト）が便利
  • TEIの分野ではOxgenが代表的なソフトだが、ライセンス費用が高いため、初学者や学術機関に所属していない人には手を出しにくい
  • Oxgenはメールアドレスを登録すれば、試用版ライセンスを使うことも可能だが、お試し期間は30日。
  • VSCodeは様々な拡張機能をインストールすることで機能追加ができるが、XML編集用、TEI/XML編集用の拡張機能も開発・公開されている。
  • 今回の作業：VsCodeでTEI/XMLファイルが編集可能な環境を整える。

手順

  1. VSCodeのインストール

    https://code.visualstudio.com/download

  1. 拡張機能のインストール
    • TEI Japanese Editor
      • https://marketplace.visualstudio.com/items?itemName=ldas.vscode-japanese-tei
    • Scholarly XML
      • https://marketplace.visualstudio.com/items?itemName=raffazizzi.sxml
    • Auto Close Tag
      • https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-close-tag
    • XML Tools
      • https://marketplace.visualstudio.com/items?itemName=DotJoshJohnson.xml
    • htmltagwrap
      • https://marketplace.visualstudio.com/items?itemName=bradgashler.htmltagwrap
    • Auto Rename Tag ★任意
      • https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag )
    • Close HTML/XML tag★任意
      • https://marketplace.visualstudio.com/items?itemName=Compulim.compulim-vscode-closetag
  1. 設定可能な項目
    • TEI japaneseEditor
      • フォントサイズ、ODDファイル、適用するCSSファイル等
    • 拡張機能を右クリック＞拡張機能の設定を開く
    • ODDファイル：vs-code-japanese-teiを選択※表示が縦書きになる
  1. 日本語スキーマファイルの作成と適用
    • TEIスキーマ
      • TEI/XMLのタグのルールを記述した「スキーマ」ファイルを定義し、それを読み込むことで使用できるタグの候補や解説を表示することができる。
    • Romaの解説
      • TEIのスキーマファイルを編集するためのウェブアプリケーション
    • 手順
      • https://romabeta.tei-c.org/にアクセス
      • TEI_ALL（customize by reducing TEI）を選択し、STARTを押下
      • Language of elements and attributes　と　Documentation Language　を日本語にする
      • 「RELAX NG schema」を選択し、ファイルをダウンロード。任意の場所に置く。
  1. ユーザスニペット
    • 「ファイル→ユーザ設定→ユーザー スニペットの構成」で任意のファイル名（「teiall.json」）を入力して新しいファイルを作成する。
    • 2行目<?xml-model href="tei_all.rng" schematypens="http://relaxng.org/ns/structure/1.0" type="application/xml"?>,”のhrefにはスキーマファイルのパスを入れる
	"TEI P5 All": {
		"prefix": "teiall",
		"body": [
			"<?xml version=\"1.0\" encoding=\"UTF-8\"?>",
			"<?xml-model href=\"tei_all.rng\" schematypens=\"http://relaxng.org/ns/structure/1.0\" type=\"application/xml\"?>",
			"<TEI xmlns=\"http://www.tei-c.org/ns/1.0\">",
			"  <teiHeader>",
			"      <fileDesc>",
			"         <titleStmt>",
			"            <title>${1:Title}</title>",
			"         </titleStmt>",
			"         <publicationStmt>",
			"            <p>${2:Publication Information}</p>",
			"         </publicationStmt>",
			"         <sourceDesc>",
			"            <p>${3:Information about the source}</p>",
			"         </sourceDesc>",
			"      </fileDesc>",
			"  </teiHeader>",
			"  <text>",
			"      <body>",
			"         <p>${4:Some text here.}</p>",
			"      </body>",
			"  </text>",
			"</TEI>"
		],
		"description": "Oxygen-like TEI (P5) All Boilerplate"
	}

動作確認

  • 「ファイル」＞「新しいテキストファイル」でxmlファイルを作成→名前を付けて保存する。
  • 「teiall」と入力し、5で作成したユーザスニペットが呼び出せることを確認する。
  • 「<」と入力し、候補のTEIタグと日本語の説明文が表示されることを確認する。
  • TEIJapaneseEditorの機能
    • ODDファイルに応じて縦書き/横書きに表示が切り替わる
    • コマンドが使えるか
      Generate panel / プレビューを開くctrl-k v
      Insert ruby / ルビを挿入するctrl-k r
      Insert warichu / 割注を挿入するctrl-k w
